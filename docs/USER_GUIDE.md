# دليل المستخدم — منصة بصيرة (Basira)

> دليل سريع للمستخدم اليومي لمنصة بصيرة

## بدء الاستخدام السريع

### 1. الدخول للنظام

```
1. افتح المتصفح
2. اذهب إلى: https://basira.yourcompany.com
3. أدخل بيانات الدخول
4. ستظهر واجهة الدردشة
```

### 2. كتابة أول سؤال

```
اكتب في مربع الدردشة:
"ما هي مبيعات اليوم؟"
```

### 3. فهم النتيجة

كل إجابة تحتوي على:

| العنصر | الوصف |
|--------|-------|
| **الإجابة** | التحليل والتوصيات بالعربي |
| **الوكيل** | أي وكيل أجاب (تحليلي/خدمة عملاء/عمليات) |
| **المصدر** | من أين جاءت البيانات |
| **الوقت** | مدة المعالجة |

## ميزات متقدمة

### تصدير التقارير

بعد الحصول على تحليل، يمكنك:

```
1. اضغط على زر "تصدير التقرير"
2. اختر الصيغة:
   - PDF (للطباعة)
   - Excel (للتحليل)
3. سيتم تحميل الملف تلقائياً
```

### تتبع سياق المحادثة

```
بصيرة تتذكر سياق المحادثة:
أنت: "ما هي مبيعات الرياض؟"
بصيرة: [إجابة]
أنت: "والأسبوع الماضي؟"  ← بصيرة تفهم أنك تسأل عن مبيعات الرياض الأسبوع الماضي
```

### التصعيد للمسؤول

```
إذا احتجت مساعدة بشرية:
• بصيرة ستخبرك عندما تحتاج قرار بشري
• يمكنك طلب التصعيد باستخدام:
  "أحتاج التحدث مع مسؤول"
  "أريد تصعيد هذه الحالة"
```

## نصائح للاستخدام الأمثل

### ✅ افعل

- **كن محدداً**: اذكر اسم الفرع أو المنتج أو التاريخ
- **اسأل بالعربي**: بصيرة مصممة للغة العربية
- **تابع السياق**: يمكنك طرح أسئلة متابعة
- **اطلب التوضيح**: إذا لم تفهم الإجابة، اسأل مرة أخرى

### ❌ لا تفعل

- **لا تسأل أسئلة مالية حساسة**: الخصومات الكبيرة والإلغاءات تحتاج موافقة
- **لا تشارك معلومات حساسة**: بصيرة لا تحتاج كلمة المرور أو بيانات الدخول
- **لا تتسرع**: بعض الاستعلامات قد تستغرق 10-15 ثانية
